## 1. 📝 概述

- deepwiki
  - deepwiki 是一个 GitHub 上大部分知名开源项目的百科全书。
  - 使用非常简单，只需要将地址中的 `github` 改为 `deepwiki` 即可。比如：
    - https://github.com/microsoft/vscode
    - https://deepwiki.com/microsoft/vscode
